📍 REGISTRO CNAME (WWW - www.ordrx.ai)
   
   Adicione este registro CNAME na GoDaddy:
   
   ┌─────────┬────────┬──────────────────────────┬────────┐
   │ Tipo    │ Nome   │ Valor                    │ TTL    │
   ├─────────┼────────┼──────────────────────────┼────────┤
   │ CNAME   │ www    │ SEU-USUARIO.github.io    │ 1 hora │
   └─────────┴────────┴──────────────────────────┴────────┘
   
   ⚠️  Substitua "SEU-USUARIO" pelo seu username do GitHub!

═══════════════════════════════════════════════════════════════

✅ DEPOIS DE CONFIGURAR:

1. Aguarde 10-30 minutos (propagação DNS)
2. Teste: dig ordrx.ai +short
3. Ative HTTPS no GitHub Pages (Settings → Pages)
