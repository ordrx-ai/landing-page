# Prompts para Geração de Ícones - Seção Pain Points

Use estes prompts no Nano Banana (ou ferramenta similar) para gerar ícones profissionais que substituirão os emojis na seção "Você Está Perdendo Dinheiro".

## Especificações Técnicas
- **Formato**: PNG com fundo transparente
- **Tamanho**: 512x512px (será redimensionado para 80x80px no site)
- **Estilo**: Moderno, minimalista, flat design com gradientes sutis
- **Cores**: Usar paleta ORDRX (roxo #a855f7, teal #00d4ff, laranja #ff6b35, azul escuro #1a1f3a)

## Diretrizes de Design

### Estilo Visual
- **Flat Design**: Sem sombras 3D, linhas limpas
- **Minimalista**: Foco no essencial, sem elementos desnecessários
- **Moderno**: Estilo contemporâneo, não vintage ou clássico
- **Profissional**: Adequado para contexto B2B/restaurantes

### Cores por Ícone
1. **Garçom Extra**: Gradiente roxo (#a855f7) → roxo escuro
2. **Pedido Errado**: Gradiente laranja (#ff6b35) → laranja escuro
3. **Mesa Parada**: Gradiente teal (#00d4ff) → ciano escuro
4. **Lucro Invisível**: Gradiente roxo (#a855f7) → teal (#00d4ff)

### Elementos a Evitar
- ❌ Emojis ou estilo cartoon
- ❌ Sombras 3D pesadas
- ❌ Cores muito saturadas ou neon
- ❌ Elementos muito complexos
- ❌ Texto dentro do ícone

### Elementos Recomendados
- ✅ Silhuetas simples
- ✅ Formas geométricas
- ✅ Símbolos universais (dinheiro, tempo, erro, dados)
- ✅ Linhas limpas e definidas
- ✅ Gradientes sutis

## Após Gerar os Ícones

1. Salvar os arquivos como:
   - `pain-icon-waiter.png`
   - `pain-icon-wrong-order.png`
   - `pain-icon-idle-table.png`
   - `pain-icon-invisible-profit.png`

2. Colocar na pasta `/images/`

3. Atualizar o HTML em `index.html` para usar as imagens ao invés de emojis
